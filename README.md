# LeetCode

Daily LeetCode practice solutions.

## Structure

- Each problem has its own folder.
- Folder format: `9-palindrome-number`
- Solution file format: `9-palindrome-number.cpp`
- Each folder includes a short `README.md` with the problem link, examples, approach, and complexity.

Example:

```text
9-palindrome-number/
├── 9-palindrome-number.cpp
└── README.md
```

## Daily Workflow

```bash
git status
git add .
git commit -m "Add solution for problem <number>"
git push
```

## VS Code LeetCode Extension

The VS Code LeetCode extension is configured to save new problems inside this repo using the same folder structure:

```text
13-roman-to-integer/
├── 13-roman-to-integer.cpp
└── README.md
```

Click `Code Now`, solve the problem, then commit and push.

## Naming Checklist

1. Use the original LeetCode problem number.
2. Use the lowercase LeetCode slug.
3. Keep the folder name and solution file name consistent.
